Ros学习的历程

## 第一周视频学习至 p.56
一开始，粗略的了解了ros的相关概念及其作用，然后在ubuntu20.04系统下比较顺利的完成安装noetic版本及其相关配置。

然后了解了launch文件的使用，自己的理解就是launch相当于一个脚本的作用，当运行launch文件时，可以一次执行多个任务节点，能够简便我们的操作。
